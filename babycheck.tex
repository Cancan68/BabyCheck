  \part*{Description}
  \vskip 1cm
  \begin{center}
    \mbox{%
      \hspace{4pt}
      \fbox{\includegraphics[height=5.644cm]{babycheck.jpg}}%
      \hspace{4pt}
      }\par
  \end{center}
  \textbf{BabyCheck} is an application designed to administrate the information of your babies.\\
  It features:\par
  \begin{itemize}
    \item unlimited number of babies
    \item birth information (weight, height, date and time)
    \item weight and height entry
    \item graphical growth follow up for the first three years
    \item list of the first words
    \item list of the first actions
    \item vaccination checklist
    \item zodiacal sign and period
    \item firstname symbolic (lucky number and period)
  \end{itemize}
  \vskip 1cm
  It works in Kg or Lb, Cm or In.\\ \\
  It works on color and black\&white devices.\\ \\
  The graphics are displayed in 320x320 mode for the SONY Cli\'es and Palm OS5 devices (if high density is available)\\ \\
  The program is \textbf{Freeware}.

  \part*{Requirements}
  \vskip 1cm
  You need at least Palm OS 3.5 to run this application.\\
  \part*{Installation}
  \vskip 1cm
  The zip file contains:\\ \\
  \begin{tabular}{l l}
    babycheck.prc & English version \\
    babycheckFr.prc & French version \\
    babycheckEs.prc & Spanish version \\
    babycheckDe.prc & German version \\
  \end{tabular}
  \vskip 1cm
  Install one of those files on the PDA.\\ \\
  Users of Microsoft Windows and the Macintosh should use the Install Tool that comes with the Palm Desktop software to install one of the above PRC files. Afterwards, initiate a HotSync to complete the installation.\\ \\
  UNIX users should use a program like "pilot-xfer" which is a part of the pilot-link package to install one of the above PRC files. (Other packages like KPilot, JPilot, and gnome-pilot will work just as well.)
